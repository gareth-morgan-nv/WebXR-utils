# WebXR utils

Some simple test pages and utils for WebXR.

## GitHub Pages

1. Repo **Settings** → **Pages**
2. **Build and deployment** → Source: **Deploy from a branch**
3. Branch: **main** (or your default branch), folder: **/ (root)**
4. Save. The site will be at `https://<user>.github.io/<repo>/` with the landing page at `/` and the pass-through demo at `/PassThrough/`.

The repo includes a `.nojekyll` file so static assets are served as-is.
